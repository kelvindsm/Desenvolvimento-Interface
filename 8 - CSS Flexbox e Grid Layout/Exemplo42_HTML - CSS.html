O que é o grid layout

O grid layout é bidimensional, permitindo que o desenvolvedor possa separar o layout da página em linhas e colunas, diferentemente do flex box que é unidimensional, ordenando os elementos em linha.

Flexbox ou Grid layout

O flexbox e o CSS Grid Layout podem ser utilizados em conjunto. você pode aplicar o flexbox dentro da grade criada pelo grid layout, aproveitando o melhor das duas técnicas.

A escolha entre flexbox e grid layout não se trata de um substituir o outro, mas sim de entender as necessidades específicas do seu layout e escolher a técnica que melhor atenda as suas necessidades.
