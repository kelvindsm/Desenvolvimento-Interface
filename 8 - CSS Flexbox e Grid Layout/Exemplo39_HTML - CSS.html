Exemplo 39 Flexbox / Flexwrap

Flex-wrap: wrap

A propriedade CSS flex-wrap é aquela que define se os itens flexíveis são forçados a ficarem na mesma linha ou se podem ser quebrados em várias linhas.

O valor padrão do flex-wrap é nowrap, o que significa que os itens flexíveis são forçados a ficarem na mesma linha, mesmo que isso signifique que eles sejam redimensionados para caber no espaço disponível. Se o valor for wrap, os itens flexíveis podem ser quebrados em várias linhas.

O flex-wrap também controla a direção em que as linhas são empilhadas. O valor wrap padrão empilha as linhas da esquerda para a direita, se o flex-direction for row, ou de cima para baixo se ele for colunm. O valor wrap-reverse empilha as linhas da direita para a esquerda ou de baixo para cima, respectivamente.
