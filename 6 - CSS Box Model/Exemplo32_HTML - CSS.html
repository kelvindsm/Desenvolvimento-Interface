Exemplo 32 - CSS Unidades de medida

Unidades de medida em CSS determinam tamanhos, espaçamentos, larguras, alturas, tamanhos de fontes, etc. Elas dizem "o quão grande ou o quão longe" um elemento pode ser. Neste caso, temos duas unidades básicas de medida, são elas: Absolutas e relativas.

Unidade de medida Absoluta

Essas unidades de medida tem como caracteristica sempre apresentarem o mesmo tamanho idependente do contexto. Neste exemplo, estamos usando no texto desta div a unidade de medida paica -pc.

Unidades de medida relativas

As unidades de medidas relativas são aquelas que dependem de algum valor externo (como fonte, tamanho da página, etc.), ou seja do tamanho do elemento pai.

Podemos exemplificar as seguintes unidades de medida relativas: em (baseado no tamanho médio do m), rem (baseado no root - r, e no em), nestes dois exemplos, podemos dizer que 1em = 1rem = 16px.
