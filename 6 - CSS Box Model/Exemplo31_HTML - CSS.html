Exemplo 31 - CSS / Background

Background refere-se em HTML ao fundo dos elementos. Ele pode ser aplicado de diferentes formas, por exemplo: mudando a cor de fundo, colocando imagens em diferentes posições e tamanhos.

Background image

Neste caso a imagem é usada como plano de fundo do elemento. Neste exemplo vamos usar uma imagem nesta div como fundo da div. O background image pode ser usado para personalizar toda a página ou apenas parte dela.

Background repeat

A propriedade background é utilizada de forma que se tenha o controle de repetição da imagem nos eixos x e y.
