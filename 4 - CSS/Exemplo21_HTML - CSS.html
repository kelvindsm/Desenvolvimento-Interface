Exemplo 21 HTML CSS

CSS é o acrônimo para folhas de estilo em cascata. Essa técnica visa personalizar, estilizar um documento HTML. Para isso, usamos regras CSS, que representam uma estrutura que define como os elementos selecionados devem ser estilizados. Essa regra é composta pelas seguintes partes:

  • seletor: indica quais elementos serão efetados pelo estilo definido.
  • Bloco de Declaração Neste caso o bloco é composto por propriedades e valores, estes elementos são aplicados aos elementos HTML.

Seletores de tipo

Como vimos anteriormente, os seletores são padrões usados para selecionar elementos HTML e consequentemente aplicar regras de estilo.

O seletor de tipo é caracterizado por utilizar nomes de tags HTML. Sendo aplicado a um ou mais tipos ou elementos de referencia.

Neste exemplo, estamos utilizando seletores de tipo para estilizar o documento HTML. Para mais informações, acesse a estrutura dessa página.
