Exemplo 23 HTML - CSS Seletores de ID

O seletor de ID é aquele que aplica a regra CSS a um componente HTML que tenha um atributo ID no componente HTML.

Esse seletor de ID deve ser aplicado a um unico elemento HTML, haja vista, que em tese ele deve ser único. Além disso, esse seletor tem prioridade máxima sobre os demais seletores (classe e tipo) por exemplo.

Neste exemplo, estamos utilizando seletores de ID para estilizar o documento HTML atual.
