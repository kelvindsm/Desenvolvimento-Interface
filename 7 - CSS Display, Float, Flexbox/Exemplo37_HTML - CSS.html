Exemplo 37 - Flex box

O flexbox é um dos sistemas mais importantes para criar layouts responsivos modernos. Ele permite que criemos layouts complexos de formas simples.

Para utilizá-lo precisamos usar a propriedade display com a opção flex. Neste exemplo, usamos alguns recursos fundamentais da sua estrutura.

Neste exemplo, mostraremos a utilização das propriedades flexdirection que possui propriedades que permitem que um elemento possa organizado no sentido horizontal ou vertical.
