Exemplo 26 CSS - Tipos de vinculação de arquivos CSS

Existem três tipos básicos de vinculação de arquivos CSS, são elas: as vinculações externas e as internas, que se dividem em duas incorporadas e inline

Neste exemplo, nós estamos usando a vinculação externa, que é a mais indicada para projetos maiores.

Para vincular arquivos CSS de forma externa nós usamos a tag link.

Exemplo de aplicação da vinculção externa.

Como diz acima, estamos vinculando um arquivo CSS externo. Esse arquivo foi/será criado na mesma pasta onde está o documento HTML, caso o exemplo 26.
