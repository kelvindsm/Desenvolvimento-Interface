Exemplo 30 HTML - CSS Cores

Quando falamos de cores em CSS, estamos falando de uma infinidade de aplicações, que podem ser definidas de diversas formas. A seguir vamos descrever cada uma delas.

RGB/RGBA

Essas cores são baseadas na combinação de cores obtidas a partir do vermelho; verde; azul, Além disso, podemos ainda adicionar uma nova variante "a", o "a" se refere ao nível de transparência, e vai de 0 a 1. Quanto mais próximo do 1, menos transparente ele será.

Cores HEX

No caso das cores baseadas no sistema hexadecimal, as cores são formadas combinando o # com numeros que vão do 0 ao 9 e da letra "a" até a letra "f". O sistema hexadecimal é muito comum em documentos HTML/CSS.

HSL/HSLA

O sistema de cores HSL é aquele que se baseia na matiz - HUE e é fornecido em graus no círculo cromático. Temos ainda o S de Saturação e o L de luminosidade.
