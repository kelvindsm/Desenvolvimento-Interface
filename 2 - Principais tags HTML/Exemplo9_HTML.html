Exemplo 9 HTML - Web semântica e links

A tag semântica <header> se refere ao cabeçalho do nosso documento HTML. Este elemento semântico é utilizado para definir uma seção introdutória, geralmente nele teremos tags de seção.

A tag semântica <nav> é usada para definir áreas de navegação dentro de uma página HTML. Ela geralmente contém menus de navegação. Abaixo exemplificamos um menu que será criado com a tag <a> representa um link de um documento qualquer. Essa tag é muito importante para realizar o roteamento de diversos hipertextos, hiperlinks, etc.

O elemento HTML main é uma tag semântica que define o conteúdo principal de uma pagina. Ela deve conter o conteúdo mais relevente para o usuário.

HTML Semântico

O html semântico é o conceito do html que se refere ao uso de elementos que possuem significado claro e descritivo, tanto para desenvolvedores quanto para navegadores. Além disso, eles são essenciais para os mecanismos de busca e leitores de tela.

Neste exemplo estamos utilizando a tag semântica section, essa tag agrupa contéudo relacionados dentro da tag main. Dentro da tag section poderemos ter também a tag article, essa tag representa um conteúdo independente, como posts de blog e notícias.

Tag de valor semântico neutro <div>

A tag <div> é aquela que possui o valor semântico neutro. Ela é muito utilizada para agrupar elementos sem um significado específico. Geralmente a utilizamos quando queremos criar um layout.

Tag <article>

A tag semântica <article> é utilizada para definir conteúdos independentes e autossuficientes.

Neste bloco, estamos usando a tag aside, que é utilizada para representar conteúdos complementares ou secundários dentro de uma página. Nessa tag podemos ainda criar menus auxiliares e ou anuncios diversos.

Neste ultimo bloco, estamos usando a tag semântica footer que é utilizada para representar o rodapé de uma página. Ele geralmente traz informações sobre direitos autorais, informações de contato, créditos pessoais, etc...
