Exemplo 11 HTML - Elemento tabela - uso de tags semânticas e de mesclagem

Para criarmos tabelas em documentos html devemos usar a tag <table>, essa tag permite que possamos organizar nossos dados em linhas e colunas. Cada tabela é composta por uma estrutura de linhas, cabeçalho e colunas(células).

Quando falamos em tags de tabelas que possuem valor semântico estamos falando das tags que vão ser interpretadas e utilizadas para indexar conteúdo tabular do documento HTML

TAGS <thead>, <tbody> e <tfoot>

As tags seânticas listadas acima, são utlizadas para organizar melhor a tabela. Elas possuem valor semântico e vão auxiliar juntamento com as outras tags semânticas a indexação pelos navegadores.
