Exemplo 8 HTML - Uso dos elementos de imagem e vídeo (img), vídeo e iframe.

Elemento de imagem img

O elemento html que representa imagens é a tag <img>, essa tag é usada para exibir imagens. Para utilizá-la podemos usar imagens dos mais variados formatos, para isso devemos usar os seguites atributos: src, que representa o caminho da imagem e alt que representa o texto alternativo, que será carregado quando a imagem não estiver disponível.
