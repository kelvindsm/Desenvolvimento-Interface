Exemplo 6 HTML - Uso das tags para formatação de texto e organização visual.

Esse exemplo demonstra o uso das tags para formatação de texto e consequentemente de sua organização visual.

Descrição

As tags abordadas são:

  • <p >: Define um parágrafo.
  • <hr >: Cria uma linha horizontal para separar conteúdos.
  • <br >: Insere uma quebra de linha.
  • <strong >: Define texto em negrito - com importância semântica.
  • <em >: Define o texto em itálico.
