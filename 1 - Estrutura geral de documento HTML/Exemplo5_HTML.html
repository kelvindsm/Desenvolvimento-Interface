Exemplo 5 - Elementos HTML de Título/seção

Este exemplo mostra como usar tags de título que podem variar do h1 ao h6.

Descrição

As tags de título são usadas para definir cabeçalhos e subtítulos. Elas variam do h1 (mais importante) ao h6 (menos importante).

Exemplo

Título principal - Mais importante

Subtítulo - h2

Subtítulo - h3

Subtítulo - h4

Subtítulo - h5
Subtítulo - h6

Neste exemplo, mostramos como usar os diferentes títulos e diferentes níveis em HTML.
