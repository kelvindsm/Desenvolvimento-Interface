Exemplo 7 HTML - Listas ordenadas <ol> e não ordenadas <ul>

Este exemplo demonstra como criar listas ordenadas e não ordenadas, incluindo subitens (listas aninhadas). As listas são utilizadas para descrever de forma de bullets ou em ordem numérica, ou alfabética de elementos

Lista ordenada - <ol>

A lista ordenada é aquela que é definida por meio de numeros, letras ou algarismos romanos.

A seguir, demonstramos uma lista ordenada que usa as tags ol e li, sendo li o item da lista.

Exemplo 1 - Lista ordenada

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5

Exemplo 2 - Lista ordenada

  1. Item 1
  2. Item 2
    1. subitem 2.1
    2. subitem 2.2
    3. subitem 2.3
    4. subitem 2.4
  3. Item 3
  4. Item 4
  5. Item 5
    1. subitem 5.1
    2. subitem 5.2
    3. subitem 5.3
    4. subitem 5.4

Lista não ordenada - <ul≷

As listas não ordenadas são aquelas onde os itens não aparecem com uma ordem específica. Essas listas utilizam marcadores como circulos e quadrados

As listas não ordenadas também são muito utilizadas na criação de menus.

Exemplo de lista não ordenada

  • Item 1
  • Item 2
  • Item 3
    • Subitem 3.1
    • Subitem 3.2
    • Subitem 3.3
  • Item 4
  • Item 5
