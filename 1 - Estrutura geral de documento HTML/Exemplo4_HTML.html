Exemplo 4 - HTML (Estrutura Básica de um documento HTML)

Este exemplo mostra a estrutura mínima de um documento HTML. Aqui estão as principais tags utilizadas:

  • <!DOCTYPE html>: Define o tipo / versão do documento html. No nosso caso, a versão atual é o html 5
  • <html html>: É o elemento central do documento, é o conteinter raíz de toda a página.
  • <head>: Contém metadados , como o título da página e links para arquivos externos.
  • <body>: Parte visível da página. Tudo que estiver dentro dessa tag ficará visível
